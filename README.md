# Math into LaTeX: Essentials

This repository is a companion resource for the book  
**Math into LaTeX: Essentials – A Complete Beginner’s Guide**.

## Purpose
- Demonstrate clean LaTeX math examples
- Provide tested sample documents
- Show reproducible LaTeX workflows
- Maintain an update log

## What this repository is
- A learning companion
- A source of verified examples
- A demonstration of professional LaTeX practices

## What this repository is NOT
- The full book source
- A free replacement for the book

Extended chapters, structured exercises,
